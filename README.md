# Rocket Trajectory Simulator

Self-built physics simulation of a multi-stage-capable rocket using first-principles physics.

## Features
- Numerical integration with `scipy.solve_ivp`
- Variable gravity
- Exponential atmosphere model for drag
- Burn time optimization using `scipy.optimize`
- Realistic parameters inspired by heavy-lift vehicles

## Results
- **Best burn time**: 300 seconds
- **Maximum velocity**: 6.908 km/s
- **Maximum altitude**: 2267 km

Achieved near-orbital velocity in a single-stage simulation in under one week of self-directed learning.

## Technologies
- Python, NumPy, SciPy, Matplotlib

## Goal
Demonstrating rapid learning and first-principles engineering ability for SpaceXAI.
